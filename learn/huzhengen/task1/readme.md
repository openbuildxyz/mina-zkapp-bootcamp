2. 概述递归零知识证明在 Mina 共识过程中的应用

递归零知识证明允许将多个交易的证明合并为一个单一的证明。

由于递归零知识证明的存在，Mina 的全节点可以在不需要存储整个区块链历史的情况下运行。

递归零知识证明确保每个新生成的区块都能验证前一个区块的有效性，从而增强了网络的安全性和一致性。
